#Project "newworldhoge"

Last Updated:2014/1/14

##About This Project
人工知能を用いて人間との対話ができるシステムを開発するために、研究を行うプロジェクトです。  
人間同士の対話により近づいた対話ができるように、人間からの入力に対し、人間から見てより自然な返答ができるようなシステムを開発することを目標としています。
